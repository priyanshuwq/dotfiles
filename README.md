##  What's Inside

- **Shell**: Fish with Starship prompt
- **Terminal**: Alacritty & Ghostty & Kitty
- **Desktop**: Hyprland with Waybar
- **Editor**: Neovim configuration
- **Notifications**: Mako
- **Wallpapers**: [Collection](https://github.com/priyanshuwq/Wallpapers.git)
- **Launcher**: Walker
- **System Info**: Fastfetch

---

##  Quick Start

```bash
# Clone the repository
git clone https://github.com/priyanshuwq/dotfiles.git ~/.dotfiles

# Navigate to the directory
cd ~/.dotfiles

# Create symlinks (customize as needed)
ln -sf ~/.dotfiles/.config/* ~/.config/
```
